Change your jQuery version and the CDN of jQuery just by onclick!

== Description ==

Want to use higher version of jQuery?

Want to use a custom CDN of jQuery?

Use this plugin now!

This plugin allow you to change your blog's jQuery version and the CDN of jQuery.

Everything you need to do after installing the plugin is only set the version/CDN of jQuery in the Option Page!

== Installation ==

###Updgrading From A Previous Version###

To upgrade from a previous version of this plugin, delete the entire folder and files from the previous version of the plugin and then follow the installation instructions below.

###Installing The Plugin###

Extract all files from the ZIP file, making sure to keep the file structure intact, and then upload it to `/wp-content/plugins/`.

This should result in the following file structure:

`- wp-content
    - plugins
        - switch-jq-version
            - lang
                | readme.txt
                | switch-jq-version-zh_CN.mo
                | switch-jq-version-zh_CN.po
                | switch-jq-version-zh_TW.mo
                | switch-jq-version-zh_TW.po
                | switch-jq-version.pot
            | LICENSE
            | license.txt
            | options.php
            | readme.txt
            | switch-jq-version.php`

Then just visit your admin area and activate the plugin.

**See Also:** ["Installing Plugins" article on the WP Codex](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins)
